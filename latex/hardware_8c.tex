\doxysubsection{Detailed Description}
Implemetación de las funciones de control del ESP32. 

\begin{DoxyVerb}Este módulo implementa la inicialización y el control del LED 
\end{DoxyVerb}
 conectado al ESP32 mediante la interfaz GPIO proporcionada por ESP-\/\+IDF.

Otorga los métodos para:\+
\begin{DoxyItemize}
\item Inicializar el periférico GPIO.
\item Encender el LED.
\item Apagar el LED.
\item Ejecutar la secuencia de parpadeo determinada. 
\end{DoxyItemize}
